# 📚 Online Library System

This project is a simple React application where users can browse books, search books, filter books by category, and add new books.

## Features

* View list of books
* Search books by **title or author**
* Filter books by **category**
* View **book details**
* Add a **new book**
* Navigation using **React Router**
* State management using **Redux Toolkit**

## Technologies Used

* React
* React Router
* Redux Toolkit
* CSS
